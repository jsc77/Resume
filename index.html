API

METHOD API AUTH DESC
GET /board/{id} none 카테고리에서 해당 게시판(id) 클릭시 불러오는 board리스트
GET /board/tag none 다수의 해쉬태그로 불러오는 board리스트
GET /board/get/{id} none 카테고리로 불러온 board리스트 중에서 하나의 게시판 불러오기
POST /board/{id} USER 선택한 카테고리 안에서 board엔티티 생성
DELETE /board/{id}/delete USER 해당 board엔티티와 참조값 삭제
POST /boardImage none 게시판 텍스트 작성시 저장되는 이미지
GET /boardImage/{id} none 게시판의 텍스트 속 이미지를 불러오기
GET /category none 네비게이션 바에 표시할 카테고리 리스트
GET /category/{id} none 카테고리 리스트 안에서 하나의 데이터만 읽어들이기
GET /comment/{id} none 게시글에 접속한 뒤에 댓글을 불러오기
GET /comment/read USER 로그인 한 후에 알람에 표시될 댓글 목록
POST /comment/{id} USER 댓글 작성하기
POST /comment/{commentId}/recommendation USER 댓글에 추천하기(한 계정당 한번만 하고 한번 더 클릭시 취소)
DELETE /comment/{commentId}/delete USER 댓글 작성자가 본인의 댓글을 삭제하기
PUT /comment/alarm/{commentId} USER 유저의 알람에 올라간 댓글을 선택했을 때 읽음 처리
GET /member USER 세션에 저장된 JWT토큰으로 로그인 한 유저 불러오기
GET /member/all USER 다른 유저에게 도움 요청할 때 사용할 유저목록
GET /member/board/{memberId} none 유저가 작성한 게시글 목록
GET /member/{id} none 클릭한 유저가 주로 활동하는 프로그래밍 언어 TOP3 목록
POST /messages USER 로그인 한 유저가 다른 유저에게 도움 요청 메세지 전송하기
GET /messages/received USER 유저가 메세지 박스에 접속시 표시할 수신 메세지 불러오기
DELETE /messages/received/{id} USER 유저가 수신한 메세지를 삭제하기
GET /messages/sent USER 유저가 보낸 메세지 목록 불러오기
DELETE /messages/sent/{id} USER 유저가 보낸 메세지 목록에서 메세지 삭제하기
GET /messages/read USER 유저에게 도착한 메세지를 알람에 띄우기
PUT /messages/alarm/{messageId} USER 알람에 올라온 데이터를 읽었을 때 읽음처리
GET /tag/all none 게시글을 검색할 때 사용할 해시태그 목록
GET /api/refresh USER JWT 토큰의 유효기간이 남았을 경우 유효기간 연장
GET /api/expired USER JWT 토큰이 만료되었을 때 에러처리
